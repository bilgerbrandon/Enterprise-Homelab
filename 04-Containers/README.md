# Containers

This section documents the deployment and management of containerized services within the homelab using Docker.

## Objectives

- Prepare an Ubuntu Server VM for Docker
- Install Docker Engine from Docker's official repository
- Deploy and configure Portainer
- Host self-managed services using Docker Compose
